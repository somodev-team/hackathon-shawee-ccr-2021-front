# Tecnologias
Para desenvolvimento do Dindo, utilizamos as seguintes tecnologias no Front-end:
 - ReactJS
 - SASS
 - Bootstrap
 - Webpack

# Desenvolvendo
Para iniciar a aplicação, apenas rode o comando `npm start`. A aplicação encontra-se dentro da pasta `src`.
